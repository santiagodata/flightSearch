
INSTRUCCIONES DE EJECUCIÓN

- Crear un directorio nuevo

- Ejecutar 'cmd' o 'terminal'. Fijarse que la ruta sea la del directorio recién creado

- Clonar el repositorio del proyecto con con el comando 'git clone https://github.com/santiagodata/flightSearch.git'

- Dirigirse al directorio principal del proyecto con el comando 'cd flightSearch'

- Una vez dentro del directorio, ejecutar el comando 'npm install' para instalar dependencias

- Escribir el comando 'ng serve -o' para ejecutar la aplicación


 -----------------++++++++----------------------+++++------------------------+++++++++++------------+++++++++++++--------

REQUERIMIENTOS IMPLEMENTADOS

- Consumir la API REST

- Los campos sólo pueden recibir tres caracteres

- Los caracteres de los campos deben ir en mayúscula

- Seleccionar moneda después de obtener la ruta

- Agregar dos monedas más (COP Y EUR)

- Los campos no pueden contener el mismo valor


REQUERIMIENTOS NO IMPLEMENTADOS

- Obtener ruta (El resultado solo se muestra en consola)

- Configurar número máximo de vuelos/ruta
